BatchOperation()

Concurrent execute batch operation and call operation callback randomly or in sequence.
Random mode is for uploading.

  1. Fire user callback when the operation is done.
    Sequence mode is for downloading.
  2. Fire user callback when the operation is done and all previous operation and callback has finished.
  3. BatchOperation guarantee the user callback is fired one by one.
  4. The next user callback can't be fired util the current one complete.
